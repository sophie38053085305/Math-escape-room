Math Escape Room
by Sophie Magerramov and Evelyn Chen
This project is meant for kids learning the Pythagorean Theory, and it's meant to help kids use logic while outputting information they learned to help them remember it.
Step 1:  Click Run
Step 2: Enter your name
Step 3: Answer the questions to escape!
Tools and Technology Used: Python, ChatGPT, tkinter
We used AI to brainstorm and answer a few questions we had
